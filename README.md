# Patrones de Diseño en C#

**Autor:** Rodrigo Sotelo Rubio
**No. Control:** 21212053

Este repositorio contiene implementaciones de diferentes **patrones de diseño** utilizando el lenguaje **C#**.

Los ejemplos fueron desarrollados como prácticas académicas para comprender la aplicación de los patrones dentro de la **programación orientada a objetos**.

---

## Estructura del repositorio

### Patrones Creacionales

- Prototype
- Abstract Factory

Cada carpeta contiene la implementación del patrón correspondiente junto con su **código fuente**, **documentación** y **ejemplos de uso**.

---

## Tecnologías utilizadas

- C#
